# 🏋️ 운동 일지 디스코드 봇

Claude AI 기반 운동 코치 + 식단 어드바이저 + 자동 일지 저장 봇

## 기능
- 디스코드에서 Claude AI와 운동/식단 대화
- 대화 히스토리 유지
- `/저장` 커맨드로 오늘 일지 자동 요약 저장

## 환경변수 설정 (Railway Variables에 입력)

| 변수명 | 설명 |
|--------|------|
| DISCORD_TOKEN | 디스코드 봇 토큰 |
| ANTHROPIC_API_KEY | Anthropic API 키 |
| NOTION_TOKEN | 노션 토큰 (선택) |
| NOTION_DATABASE_ID | 노션 DB ID (선택) |

## 사용법

| 입력 | 설명 |
|------|------|
| 그냥 채팅 | AI와 운동/식단 대화 |
| `/저장` | 오늘 일지 요약 저장 |
| `/초기화` | 대화 히스토리 초기화 |
| `/히스토리` | 현재 대화 턴 수 확인 |
| `/도움말` | 도움말 표시 |
